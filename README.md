# 🛒 E-Commerce Sales Database

## 📌 Overview

This project is a SQL assignment completed as part of my internship. It demonstrates SQL concepts by analyzing an E-Commerce Sales Database using MySQL.

## 🎯 Objective

The objective of this project is to analyze sales data using SQL by performing:

- Data retrieval
- Filtering using WHERE
- Aggregation using GROUP BY
- Joins
- CASE statements
- Constraints
- Transactions
- Index optimization

## 🛠️ Technologies Used

- SQL
- Git & GitHub

## 📈 Key Business Insights

- Retrieved customer and product information.
- Filtered orders based on status, category, and date.
- Calculated total revenue and average product prices.
- Analyzed product categories and order status.
- Combined multiple tables using JOINs.
- Implemented transactions to ensure data consistency.
